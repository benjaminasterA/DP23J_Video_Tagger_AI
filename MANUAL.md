# 📖 Video Tagger AI User Manual (사용자 매뉴얼)
**Version:** Ver 1.0 | **Last Updated:** 2026.02.26 01:15:49

**Video Tagger AI**는 CLIP 모델을 활용하여 비디오 내 장면을 텍스트나 이미지로 검색하고 관리할 수 있는 도구입니다.

## 🛠️ 설치 및 실행 방법 (Installation)

1. **필수 라이브러리 설치:**
   ```bash
   pip install streamlit pandas numpy Pillow opencv-python sentence-transformers torch
   ```

2. **애플리케이션 실행:**
   ```bash
   streamlit run frontend_video.py
   ```

## 🌟 주요 기능 (Key Features)

### 1. 비디오 인덱싱 (Indexing)
- 사이드바에서 영상 파일(`mp4`, `mov`, `avi`, `webm`)을 업로드합니다.
- **AI 모델** 옵션을 선택한 후 **🚀 모델 로드 & 프레임 인덱싱** 버튼을 누르면 AI가 분석을 시작합니다.
- **AI 모델 옵션 안내:**
    1. **CLIP (openai/clip-vit-base-patch32)**
        - **특징:** 표준 CLIP 모델.
        - **장점:** 속도가 빠르고 메모리 사용량이 적어 균형 잡힌 성능 제공.
        - **용도:** 일반적인 빠른 장면 검색.
    2. **CLIP Large (vit-large-patch14)**
        - **특징:** 고정밀 대형 모델.
        - **장점:** 세밀한 분석을 통해 **검색 정확도(Accuracy)**가 매우 높음.
        - **단점:** 로딩 및 연산 속도가 상대적으로 느림.
        - **용도:** 복잡한 사물이나 정밀한 검색이 필요할 때.
    3. **Korean CLIP (실험적)**
        - **특징:** 한국어 데이터 추가 학습 모델.
        - **장점:** **한국어 검색어(Query)**의 미묘한 뉘앙스를 가장 잘 이해함.
        - **용도:** 한국어 문장형 검색어로 검색할 때 최적.

    > **💡 추천 선택 기준:**
    > - 속도 중시 → 1번 (기본 CLIP)
    > - 정확도 중시 → 2번 (CLIP Large)
    > - 한국어 자연어 검색 → 3번 (Korean CLIP)

### 2. AI 장면 검색 (AI Scene Search)
- **Text Query:** 찾고자 하는 장면을 묘사합니다. (예: "푸른 바다 위에서 서핑하는 사람")
- **Reference Image:** 이미지를 업로드하여 유사한 장면을 찾습니다.
- **🔍 RUN SEARCH** 버튼을 눌러 검색을 실행합니다.
- **결과 카드:**
    - **👉 장면 이동**: 해당 시간대로 영상 플레이어를 이동시키고 '수동 태깅' 탭으로 전환합니다.
    - **📌 태그 저장**: 검색된 핵심 정보를 즉시 저장합니다.

### 3. 수동 태깅 (Manual Tagging)
- 특정 장면에 대해 레이블(제목)을 붙여 저장할 수 있습니다.
- 저장된 태그는 **📋 태그 목록**에서 확인하고 **📥 CSV 다운로드**를 통해 저장할 수 있습니다.

## 💡 검색 팁 (Tips)
- CLIP 모델은 구체적으로 묘사할수록 더 정확한 결과를 제공합니다.
- 인덱싱은 브라우저 세션 동안 유지되므로, 다른 모델로 테스트하려면 재인덱싱이 필요할 수 있습니다.
